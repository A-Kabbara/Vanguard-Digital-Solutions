<task type="auto">
  <name>Task 1: Create css/typography.css with heading, body, and utility styles</name>
  <files>css/typography.css</files>
  <action>
Create `css/typography.css` with the following styles. All values must come from tokens -- no hardcoded colors, no hardcoded font sizes, no magic numbers.

**Heading styles (h1-h4):**
```css
h1, h2, h3, h4 {
  font-family: var(--font-heading);
  font-weight: 400;   /* Instrument Serif only has Regular 400 */
  line-height: 1.08;
  letter-spacing: -0.03em;
  text-wrap: balance;
  color: var(--color-text);
}
```

Individual heading sizes:
- `h1 { font-size: var(--fs-hero); letter-spacing: -0.04em; }` -- tighter tracking for largest size
- `h2 { font-size: var(--fs-2xl); }`
- `h3 { font-size: var(--fs-xl); letter-spacing: -0.02em; }` -- slightly looser for medium headings
- `h4 { font-size: var(--fs-lg); letter-spacing: -0.01em; }`

**Body text (p):**
```css
p {
  font-size: var(--fs-base);
  line-height: 1.65;
  color: var(--color-text-muted);
  text-wrap: pretty;
}
```
Note: Body paragraphs use `--color-text-muted` (secondary #9B9BA8) for the softer reading experience on dark backgrounds. Headings use `--color-text` (primary #F0F0F5) for hierarchy contrast.

**Lead/intro paragraph:**
```css
.text-lead {
  font-size: var(--fs-lg);
  line-height: 1.55;
  color: var(--color-text-muted);
}
```

**Small text:**
```css
small, .text-sm {
  font-size: var(--fs-sm);
}
```

**Extra small text:**
```css
.text-xs {
  font-size: var(--fs-xs);
}
```

**Mono utility class:**
```css
.mono {
  font-family: var(--font-mono);
  font-size: var(--fs-xs);
  font-weight: 500;
  letter-spacing: 0.06em;
  text-transform: uppercase;
  color: var(--color-accent-text);
}
```
Uses `--color-accent-text` (the AA-safe lightened accent) since mono labels are typically small text sized.

**Strong/bold (for Inter body text only):**
```css
strong, b {
  font-weight: 600;
}
```

**Italic handling for Instrument Serif:**
```css
h1 em, h2 em, h3 em, h4 em {
  font-style: italic;  /* Instrument Serif Italic is available */
}
```

**Link text with accent color:**
```css
.text-accent {
  color: var(--color-accent-text);
}
```

Add a top comment: `/* Typography System - Vanguard Digital */`

CRITICAL: Do NOT use `font-weight: 700` or `font-weight: bold` on any element that uses `var(--font-heading)`. Instrument Serif only has weight 400. The browser would synthesize a faux bold which looks terrible.
